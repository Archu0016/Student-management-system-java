# Student Management System

A desktop-based Student Management System developed using Java, Java Swing, JDBC, and MySQL. The application allows users to manage student records through CRUD (Create, Read, Update, Delete) operations using a graphical user interface.

## Features

- Add Student
- View Student Details
- Update Student Information
- Delete Student Records
- Search Student Records
- Sort Student Records

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- SQL

## Concepts Applied

- Object-Oriented Programming (OOP)
- CRUD Operations
- Database Connectivity
- GUI Development
- Exception Handling

## Database

The application stores student information in a MySQL database using JDBC.
